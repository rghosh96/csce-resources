basic linux commands

some basic linux commands to help you navigate your terminal or Turing environment, as well as compile c++ files. these are the ones i personally use the most, although there are more very useful ones out there! this is bare minimum to get you started.

ls

This command will list everything in the current directory.


cd [folder name]

Entering “cd” followed by a folder name will move u into that folder. For example, “cd project3” will move you into the project3 folder.

☆ note: if using windows command prompt, cd will be dir

☆ note: i HIGHLY recommend NOT using spaces in your folder names, to keep things simple


cd ..

Entering “cd” followed by two dots (there IS a space in between cd & the two dots) moves you back up one directory. For example, if you “cd ..” from inside the project4 folder, you will be back at your home directory


exit

Exits Turing and returns to ur normal terminal/command prompt.


g++ -Wall *.cpp -o output
./output

These are the big commands! Once inside your project folder, run these two commands to first compile ur program, and then run it. works if you have just one cpp file, or multiple cpp files (for classes)

☆ note: on windows command prompt, it's g++ -Wall *.cpp -o output.exe then output.exe instead
